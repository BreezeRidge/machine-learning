# 机器学习系统学习

从零开始的机器学习学习之路，理论与实践并重。

## 学习路线

| 阶段 | 目录 | 主题 | 状态 |
|------|------|------|------|
| 01 | `01-python-basics/` | Python 数据科学基础 (NumPy, Pandas, Matplotlib) | 🔲 |
| 02 | `02-math-foundations/` | 数学基础 (线性代数, 概率统计) | 🔲 |
| 03 | `03-supervised-learning/` | 监督学习 (回归, 分类) | 🔲 |
| 04 | `04-unsupervised-learning/` | 无监督学习 (聚类, 降维) | 🔲 |
| 05 | `05-model-evaluation/` | 模型评估与优化 | 🔲 |
| 06 | `06-neural-networks/` | 神经网络入门 | 🔲 |

## 环境准备

```bash
pip install numpy pandas matplotlib scikit-learn jupyter
```
