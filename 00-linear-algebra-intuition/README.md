# 00 - 线性代数直觉（零基础起点）

## 为什么有这一章

后续模块（theory.md、数学基础、所有 ML 算法）默认你知道以下概念：

- 向量、矩阵、点积、范数
- 矩阵乘法的几何意义
- 特征值、特征向量

如果这些词对你来说完全陌生，**请不要跳过这一章**。本章不讲证明、不做习题，只建立"看到这些词能在脑子里浮现画面"的直觉。

## 学习哲学

三条原则，按重要性排序：

1. **几何直觉优先于符号**：先在脑子里看到画面，再去理解公式
2. **代码验证优先于推导**：每个概念都要用 Python 跑一次，看到结果
3. **广度优先于深度**：第一遍只求"听说过、能脑补画面"，不求"能独立证明"

## 学完后的自测

完成本章后，你应该能不看任何资料回答：

- [ ] 画出两个向量相加的几何图像
- [ ] 说出"矩阵 × 向量"的几何意义（一句话）
- [ ] 解释 2×2 矩阵的行列式为 0 意味着什么
- [ ] 在 NumPy 里手写一段代码：给定矩阵 A 和向量 v，算出 A @ v 并画出变换前后的图
- [ ] 说出特征向量的定义（不要求会算）

答不上的问题，回到对应章节重学。**每一条答不上就是一个明确的学习信号**，不是失败。
