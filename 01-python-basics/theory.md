# 向量、矩阵与数据表示

## 为什么机器学习离不开矩阵？

机器学习的核心操作可以归结为一句话：**用矩阵运算从数据中提取模式**。

一个数据集本质上就是一个矩阵：

```
        特征1  特征2  特征3
样本1 [  1.2,  3.4,  5.6 ]
样本2 [  2.1,  4.3,  6.5 ]
样本3 [  3.0,  5.2,  7.4 ]
```

这是一个 3×3 矩阵，记作 X ∈ R^(m×n)，其中 m=3（样本数），n=3（特征数）。

## 向量（Vector）

向量是一维数组，在 ML 中有两种常见含义：

1. **特征向量**：一个样本的所有特征值，如 x = [身高, 体重, 年龄]
2. **参数向量**：模型的权重，如线性回归的 w = [w1, w2, w3]

### 向量运算的 ML 含义

**点积（Dot Product）**：x · w = x1*w1 + x2*w2 + ... + xn*wn

这就是线性回归的预测公式 ŷ = x · w + b。点积衡量两个向量的"相似程度"。

**范数（Norm）**：衡量向量的"大小"

- L1 范数：||x||₁ = |x1| + |x2| + ... + |xn|（对应 Lasso 正则化）
- L2 范数：||x||₂ = √(x1² + x2² + ... + xn²)（对应 Ridge 正则化）

## 矩阵乘法的意义

矩阵乘法 Y = X @ W 的几何意义是**线性变换**——把数据从一个空间映射到另一个空间。

神经网络的每一层本质上就是：`output = activation(X @ W + b)`

```
输入 X (m×n)  ×  权重 W (n×k)  =  输出 Y (m×k)
m 个样本         n 个输入特征       m 个样本
n 个特征         k 个输出特征       k 个新特征
```

## 广播机制（Broadcasting）

NumPy 的广播规则让不同形状的数组可以做运算，这在 ML 中极其常用：

```python
# 对每个样本减去均值（数据标准化）
X_centered = X - X.mean(axis=0)  # X 是 (m, n)，mean 是 (n,)，自动广播
```

规则：从右往左对齐维度，维度为 1 或缺失的可以广播。

## 为什么 NumPy 比 Python 列表快？

1. **连续内存**：NumPy 数组在内存中连续存储，CPU 缓存命中率高
2. **类型固定**：所有元素同类型（如 float64），无需类型检查
3. **C 语言底层**：核心运算用 C/Fortran 实现，绕过 Python 解释器开销
4. **SIMD 指令**：利用 CPU 的单指令多数据并行能力

这就是为什么 ML 代码要用向量化运算而非 for 循环。
